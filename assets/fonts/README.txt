# 字体文件说明 (Font Requirements)

为了使游戏中文标题和界面呈现最佳的"像素风格"，请手动下载中文字体文件并放置在此目录中。

## 推荐字体

### 1. IPix中文像素字体 (推荐)
- **名称:** IPix (12px)
- **下载地址:** https://github.com/SolidZORO/zpix-pixel-font (或其他来源如 ipix.ttf)
- **操作:** 将下载的 `.ttf` 文件重命名为 `ipix.ttf`。

### 2. Zpix (最像素)
- **名称:** Zpix
- **操作:** 将下载的 `.ttf` 文件重命名为 `ipix.ttf` (为了匹配 CSS 配置)。

## 安装步骤
1. 下载上述任意一款像素字体。
2. 将文件放入 `assets/fonts/` 目录。
3. 重命名为 `ipix.ttf`。
4. 刷新网页即可看到中文像素效果。
