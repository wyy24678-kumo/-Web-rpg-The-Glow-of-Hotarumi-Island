# 美术素材规格说明 (Asset Specifications)

为了确保游戏在不同移动设备（从 iPhone SE 到全面屏手机）上都能获得最佳视觉体验，请遵循以下美术素材规格。

## 1. 启动封面与全屏背景 (Full Screen Backgrounds)

由于 `background-size: cover` 属性会根据屏幕比例裁剪图片，我们推荐使用 **"移动端优先"** 的长图规格。

### 推荐参数
- **分辨率 (Resolution):** `1080 x 2400 px` (9:20 比例)
  - *兼容性:* 完美适配现代全面屏手机 (iPhone 13/14/15, Android 旗舰)。
  - *最小宽度:* 1080px
- **安全区域 (Safe Zone):**
  - 请将核心视觉元素（如游戏标题位置预留、角色面部、关键道具）集中在画布中央的 **800 x 1600 px** 区域内。
  - 图片边缘（特别是上下两端和左右最外侧 10%）可能会在某些设备上被裁剪，请仅用于绘制氛围背景（如天空、地面、模糊场景）。
- **文件格式:** `JPG` (推荐, 质量 80-90%) 或 `PNG`。
- **文件大小:** 建议控制在 500KB - 1MB 以内。

### 示意图
```
+-------------------------+ 1080px
|      可被裁剪区域       |
|   +-----------------+   |
|   |                 |   |
|   |    安全区域     |   |
|   |   (核心内容)    |   | ~1600px
|   |   800x1600px    |   |
|   |                 |   |
|   +-----------------+   |
|      可被裁剪区域       |
+-------------------------+ 2400px
```

## 2. UI 元素 (UI Elements)

### 按钮与图标
- **风格:** 像素风格 (Pixel Art)
- **缩放:** 游戏会自动缩放 UI，建议原图尺寸不要过大。
- **格式:** `PNG` (透明背景)
