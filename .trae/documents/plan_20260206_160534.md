# 开发主菜单界面计划

我将按照您的要求，开发《萤见之岛的微光》的主菜单界面。

## 1. 修改 `index.html`
- **引入字体**：添加 Google Fonts 的 "Press Start 2P" 或类似像素风格字体，以确保复古视觉体验。
- **构建结构**：在 `#game-container` 中添加主菜单的 HTML 结构，包括游戏标题（作为占位，后续可替换为 Logo）和两个核心按钮（【开始旅途】、【继续旅途】）。

## 2. 样式实现 `styles/main.css`
- **背景设置**：将 `#game-container` 的背景设置为 `assets/images/001_Cover_startgame.png`，并使用 `background-size: cover` 和 `background-position: center` 确保在不同分辨率下适配。
- **像素风格按钮**：
    - 使用 CSS `box-shadow` 和 `border` 模拟像素化的立体按钮效果。
    - 设计悬停 (`:hover`) 和点击 (`:active`) 的视觉反馈（如颜色变化、按压位移）。
    - 确保按钮文字清晰可读，配合像素字体。
- **响应式布局**：使用 Flexbox 布局使菜单居中，并确保在移动端和桌面端都有良好的显示比例。

## 3. 交互逻辑 `scripts/ui.js`
- **事件绑定**：编写 `initMainMenu()` 函数，为按钮绑定点击事件。
- **预留接口**：

    - 【继续旅途】点击后调用 `loadGame()`（预留）。
- **交互反馈**：添加简单的控制台日志 (`console.log`) 以验证点击是否生效。

## 4. 核心入口 `scripts/main.js`
- **初始化**：在游戏启动时调用 `ui.initMainMenu()`，确保页面加载完成后立即显示主菜单。
